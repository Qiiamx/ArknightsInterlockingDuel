ArknightsInterlockingDuel
aka AID

干员资源提交位置: 
-ui
--public
---images 职业图标（名称与JSON文件匹配）
---icon 干员头像（名称与JSON文件匹配）
---data 干员和分支json文件

项目分为前后端
均需要node 20 +

后端启动方式
npm i
node service.js

前端启动方式
cd ui
npm i
npm run dev

部署方式
请把 deploy.sh 脚本放在你的文件夹下启动（windows请根据你的情况找AI）
默认8080端口
